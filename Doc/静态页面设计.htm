一、html的基本语法和文档结构

html即超文本标记语言hypertext markup language，最新版本是html5系列、语法上由标签和属性的键值对以及多层嵌套结构组成。比如，这是一个典型的html5结构：

<!DOCTYPE html>

<html lang="可以是zn-cn、en或者其他语言">

<head>

       <meta charset="UTF-8">

       <title>标题</title>

</head>

<body>

       正文

</body>

</html>

四、css的引入

html的历史显然要早于css；css即层叠样式表，引入css的目的是实现页面内容和表现分离、方便页面维护；css也可以在手册中找到具体的用法。

1.内联样式

比如<p style=”…”></p>

2.内部样式表

在页面<head/>中定义样式<style type=”text/css”>…</style>，…中也可以引入外部样式@import url(“…”);

3.外部样式表

在页面外定义样式、在页面中引入<link rel=”stylesheet” type=”text/css” href=”外部样式表的完整名称.css”/>

4.设置网站的图标

<link rel="shortcut icon" href="完整的图标资源名称.ico" type="image/x-icon"/>

六、css属性的运用

在参考手册中css属性有着非常详细的介绍比如光标的样式cursor，但是要考虑的是怎么用这些属性解决遇到的问题；可以先将margin、padding初始为0，再根据先后顺序写需要的选择器；通过box-sizing（有兼容性要求）设置盒式模型，怪异的盒式模型在一些情况下可能会实用些；display属性中，inline表示一行行的排下去，block是一列列的排下去；一个文字大概是1em长宽

1.布局

布局中的定位：relative相对自己原来应该在的位置、absolute相对已经定位的祖先元素（如果没有就相对<body/>做定位）、fixed相对窗口，并结合top、left等属性来定位自己的位置

(1)对标签布局

①对齐

长度一样、方向一致才有可能对齐。比如登录页面用width限定输入框提示文字的长度而且文字的方向都向右，这样在列的方向上就都能对齐。

②换行

a.用br标签，用在文字上可能会好些

b.上面内容是浮动，一个属性是clear:both;的div能够实现换行

③居中

a.“margin: 0 auto;”可以使div居中、前提是div有宽度而且不是全屏

b.“position:relative;left:50%;margin-left:div一半的距离;”也可以实现div的居中

c.“text-align:center;”标签中的行内元素（比如文字）居中，但是无法使标签本身和块级元素居中

d.设置height等于line-height，上下垂直居中，设置单行文本居中时比较常见

e.设置vertical-align实现垂直居中，前提是使用表格

④溢出处理

溢出会破坏容器和页面布局，所以一定要选择合适的处理办法，关于overfow属性在参考手册上有非常具体的介绍

(2)对页面布局

div结合float

兼容性强，缺点是div的底部不能自动对齐

div结合display:box;

盒子是垂直排列，盒子中的div水平排列，但是box有兼容性的问题

div结合columns属性

主要用于div中的文字排版，不适合运用于整个页面，columns有兼容性的问题

框架集

这种布局一般要结合javascript来使用，缺点是页面框架固定、不够灵活

3.渐变过渡

必须要设置的是过渡属性transition-property，然后选择设置过渡时间、延迟时间等属性

4.媒体查询

查询屏幕或设备显示的方向、尺寸等内容，最常用的可能是查询屏幕尺寸动态设定css样式，比如：

//对IE浏览器无效，类似的属性有min-width、max-width

@media all and (min-height:300px) and (max-height:500px){

       body{color:#f00;}

}

5.帧动画

一张图片展示不同时刻展示不同的部分，有兼容性的要求；使用animation:…;，之后定义需要的帧动画；多个动画之间的先后顺序可以通过时间的延迟的决定；如果动画的效果不好，不妨尝试其他动画过渡的函数

//ball是动画的名称，在animation属性中会用到

@keyframes ball

{

       /*定义帧中每一个时间段的css效果*/

       0% {left:0; top:0;}

       25%{left:200px; top:0;}

       50%{left:200px; top:200px;}

       75%{left:0; top:200px;}

       100%{left:0; top:0;}

}
